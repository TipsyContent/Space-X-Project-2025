Space X Project
Dette projekt er en SwiftUI-app, der viser SpaceX API, Rockets, launchpads og relaterede detaljer.

Om projektet
Appen henter data fra Space X API og viser dem i lister og på et kort. Brugere kan:
Se alle Launches missioner
Gemme favorit-lanceringer
Se detaljer om hver Launch, Rockets, launchpad og medieindhold
Navigere via et kort over launchpads
Logge ind og ud og data er gemt på account

Teknologi
SwiftUI
Firebase til brugerautentifikation
MapKit til kortvisning
AsyncImage til billedhåndtering

vedrørende udvikling
Appen er udviklet med Xcode 16.4, som var den nyeste version tilgængelig for mig
På grund af hardwarebegrænsninger (ingen Mac til rådighed) kunne jeg ikke teste MapKit-funktionaliteten direkte på min maskine.
Kortvisningen blev først testet på min ven’s Mac, da min egen løsning brugte en VM (virtual machine), som havde begrænsninger for MapKit.
Dette kan betyde, at nogle funktioner, specielt kortinteraktion, ikke kunne verificeres fuldt ud lokalt.
